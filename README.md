## A small Laravel RESTful demo with data import features

## Requires
Follow Laravel 6 min requirements.
Note: PHP version must be 7.2

## To run locally
1. Install the app
```
composer install
```

2. Run migrations
```
php artisan migrate
```

3. Run import command
```
php artisan player:import
```

4. Run dev server
```
php artisan serve
```

5. Access player api
```
http://127.0.0.1:8000/api/players
```
